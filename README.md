# doblerendija
El experimento de la doble rendija fue planteado por Thomas Young en 1801, este lo hizo con el fin de demostrar que la luz se comportaba como una onda.Este experimento se puede  evidenciar si se introduce una fuente de luz y por dos lineas paralelas pasa el haz de este.El haz debe ser mayor que ese grosor en el momento del impacto con la rendija, el haz queda dividido en dos para luego chocar con una placa(en este experimento la pared) que se encuentra despues de la rendija. Así como lo explica la imagen tomada de http://cienciacomonunca.blogspot.com/2014/02/experimento-de-la-doble-rendija.html

# Materiales:

- Un láser
- Bisturí 
- Cartón cartulina
- Papel Aluminio

# Desarrollo:
1. En el papel aluminio, se realizan dos lineas paralelas con una distancia aprox de 1mm, esto depende del haz de luz.
      ![IMG_3023](https://user-images.githubusercontent.com/77986680/110364923-60fea580-8012-11eb-8512-6cb13ae26860.jpeg)
2. Este se coloca en el cartón cartulina y así tener la doble rendija que será por donde pasa el haz de luz.
      ![IMG_3026](https://user-images.githubusercontent.com/77986680/110365050-8b506300-8012-11eb-8d1b-684f7cc728d0.jpeg)
3. colocamos nuestro láser a una distancia aprox de 30 cm de la doble rendija y lo acomodamos para que el haz de luz pase por medio de estas.
      ![WhatsApp Image 2021-03-08 at 1 41 39 PM](https://user-images.githubusercontent.com/77986680/110366404-3dd4f580-8014-11eb-928c-56b8d243546a.jpeg)

# sistema probabilístico 
El experimento sera representado como un sistema cuantico que tendra una matriz de adyacencia asociada y un vector el cual representara el estado inicial del sistema, donde sus posiciones representaran una conexion  entre componentes del sistema, a continuacion se mostrara un ejemplo del sistema cuantico para el experimento de la doble rendija y su matriz asociada, para esto, se tomó en cuenta el sistema presentado en el libro "quantum computing for computer scientists" en el capitulo 3.
